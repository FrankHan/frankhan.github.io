Jekyll加github建立网页教程

Posted on 2021-01-09 | In Diary , 前端 |

简单、模式化、同时自己修改的余地也大的网页解决方案。

对于个人来说，不像大公司那样网页跟新频率那么高，而且个人网站多数是静态网页，比如博客、展示、简历等。在这种情况下主要需要考虑撰写、代码托管：

  • 撰写：

    • wordpress、wix等这种是一类，这类是收费的。优点：这类提供了撰写＋托管的服务，基本上不懂html、css也可以实现建站。缺点：自定义（自己修改点html、css）的程度是最低的。而且网站下面基本上都有wordpress或者wix的标识，此外wix这种很难自定义域名。

    • Dreamweaver：优点：很少，上古时代的网页建站软件，不需要写html、css代码。缺点：经常排版错误、工具也特别卡，基本上是落后于时代的产品。十年前的网页三剑客，flash、dreamweaver基本上行将就木，没人学了。

    • 网上的html模板：有很多html模板商城。优点：自定义程度高，可以自行修改代码。缺点：不方便撰写，尤其是写博客等，不可能手动一行一行敲<>换行、空格标签。打开文件满屏的<>标签，影响内容写作体验。而且很多老的html页面是不支持pc、mobile端自适应的。

    • Jekyll：今天我想推荐的工具。一个Jekyll网站 = Markdown + HTML + CSS。通常，只需要在免费的网站库（如 https://jekyllthemes.io/ ）挑选好Jekyll模板，其中html、css都是写好的基本不用更改，内容创作只需要写markdown，而markdown基本就像写word或者记事本（markdown语法网上搜索就很多： https://www.cnblogs.com/xiaoliangL/p/7026394.html ），可以聚焦关注内容，不用谢html标签。而自定义的空间有很大，如果对样式不满意，可以自己修改Jekyll模板中的html或者css。

  • 托管：github或者国内的码云托管肯定是首选。Jekyll或者自己的html都可以使用github托管，全部文件上传到一个github repo、再在repo设置中开启GitHub Pages功能即可实现网址访问。

下面做一个全过程步骤示例：

1. 挑选一个自己中意的 Jekyll 模板

Jekyll模板库搜索能找到很多，比如：

  • https://jekyllthemes.io/

    点击右上角的 Browse Themes 还能看到分类的 portfolio themes、blog、GitHub Pages themes等。

  • https://jekyllrb.com/docs/themes/

  • http://jekyllthemes.org/

    我们选择一个免费的theme比如：
    https://jekyllthemes.io/theme/gridster-jekyll-theme

    点击 ‘Get on github’后到达：https://github.com/DigitalMindCH/gridster-jekyll-theme

    点击 “Code”-down zip下载全部代码。

    2. 开始一些基本设置

    网站的名字、描述可以被Google等搜索到，其设置就在下载的文件夹的 _config.yml 文件中，将这个文件用sublime或者vscode（注意不是visual studio！）等记事本编辑器打开，就可以自己修改其中的设置了：

    修改后保存。

    3. 代码托管到github

    （这里其实少了一步用nodejs生成本地内容）

    打开github网站，登陆，新建一个repo：Repository name比如写 my1jeykyll，点击 Add a README file，生成repo。

    打开 https://github.com/你的github用户名/my1jeykyll/upload/main ，把整个gridster-jekyll-theme-master文件夹里面的文件 ctrl+A全选，拖到这个网页的上传区域，就把代码托管到github了。由于文件比较多，可能需要分批选中拖过来。注意整个文件夹 _data 、 _includes都可以选中这个文件夹拖进来，注意不要改变文件树的相对路径关系。（其实这里应该用git客户端更好更自动一些）

    上传完成后在 repo 界面的settings-github pages：开启github pages，注意如果本身已经绑定了域名，https的要点击开启https。

    托管完成，可以访问了： https://frankhan.net/my1jeykyll/

4. 修改网页具体内容

Jekyll书写的内容一般都很好找，比如：

  • 网站首页的post都在：gridster-jekyll-theme-master_posts

  • 图片都放置在： gridster-jekyll-theme-master\img
