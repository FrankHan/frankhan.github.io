使用 Manim 绘制三体运动视频

Posted on 2020-03-12 | In Diary , Python |

Manim安装过程中几个大坑全纪录。

视频完成效果

三体就是三颗恒星的星系，视频中三个发光的是恒星（类似太阳），小的蓝色不发光是一个行星（类似地球），构成不稳定无规律但又不太会破裂的星系。

注：浏览视频可能需要科学上网。

说明和致谢

这只是一个教如何使用的教程。本教程中涉及的所有代码来自：

  • manim ： https://github.com/3b1b/manim/
    这是一个数学可视化的Python工具。

  • cigar666 的绘制三体运动视频代码： https://github.com/cigar666/my_manim_projects
    这是cigar666写的可以绘制三体运动的代码，他做了一个非常漂亮的视频在b站：
    《三体运动模拟（含行星） | python+manim制作》：https://www.bilibili.com/video/av94380754/

在此致谢。下面这个教程目的是介绍如何上述工具具体怎么用。

环境说明

本人采用的是

Step 1: 安装Python和Pip

安装Python3.7，注意只有3.7才行。一个电脑上通常可以安装多个版本的Python，不过也很容易乱。使用Manim时务必确保 Shell或命令行中运行 python时是3.7版本。

注意安装64bit的！

一步一步安装python图文教程：https://www.jianshu.com/p/d999b558bdda

安装pip。如果按照上个图文教程，已经安装了pip（推荐）。

Step 2: 安装Manim

注意，如果是windows，一定不要按照Manim官网上的默认安装，要用Directly (Windows)方式：

https://github.com/3b1b/manim/#directly-windows

1、安装FFmpeg，一定要按照提示安装https://www.wikihow.com/Install-FFmpeg-on-Windows
安装完之后有个test，也完全通过。

2、手动安装 Cairo, https://www.lfd.uci.edu/~gohlke/pythonlibs/#pycairo
pycairo‑1.18.0‑cp37‑cp37m‑win32.whl已经没了，亲测可用pycairo-1.19.1-cp37-cp37m-win_amd64.whl:

3、 LaTeX 不涉及生成 pdf等的话不用装。

4、 SoX. 按照官网说明即可。就几MB很小。

5、安装 Manim。建议在D盘新建文件夹

然后 shift+右键 在此文件夹处调出 Shell：

在最后一步会安装requirements.txt中的所有依赖，可以看到pycairo已经安装并符合要求。如果没装pycairo那么会遇到后面写的大坑的一个。

注意：pip3 install -r requirements.txt 时最好连接科学上网，不然会出现 Timeout报错。

Step 3: 安装完毕，使用 Manim

此时已经安装完了。

在 D:\manim\manimshift+右键 在此文件夹处调出 Shell，运行Manim的示例：

说明：

  1. 有可能是python3，取决你安装python的方式。
  2. 手动安装的manim必须使用 python manim.py example_scenes.py，如果是pip安装成功的直接 Manim example_scenes.py。
  3. SquareToCircle 是 Manim example_scenes.py中的一个Class，实际上这个py文件中有很多例子，SquareToCircle是其中一个例子。
  4. -pl是 preview ，low画质的意思。

如果运行成功，说明你已经安装 Manim 成功了。会自动跳出预览视频，视频会自动存在media文件夹里。
